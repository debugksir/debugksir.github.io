ubantu上安装mysql

2019-03-09
环境配置
  • sudo apt-get install mysql-server
  • apt-get isntall mysql-client
  • sudo apt-get install libmysqlclient-dev

若安装过程提示设置root密码，记住密码；如果没有的话使用命令：set password for 'root'@'localhost' = password('yourpassworld');， 或者 mysqladmin -uroot password [password

  • 查看是否安装成功： sudo netstat -tap | grep mysql
  • 启动： service mysql start
  • 停止： service mysql stop
  • 重启： service mysql restart
  • 登录： mysql -u root -p

数据库可视化工具推荐mysql-workbench

创建用户预授权

  1. 登录数据库：
    mysql -uusername -p

  2. 创建新的登录用户：
    create user newname identified by 'password';

  3. 远程登录授权
    grant all privileges on *.* to 'newname'@'%' identified by 'password' with grant option;

  4. 刷新
    flush privileges;

  5. 创建数据库
    create database 'databasename'
