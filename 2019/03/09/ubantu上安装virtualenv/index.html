在ubantu上安装virtualenv

安装命令： pip install virtualenv
查看是否安装成功： virtualenv --version

使用:

mkdir demo-flask (创建目录)
cd demo-flask （change到目录）
virtualenv venv (ps: venv 命名可以自定义, 习惯上用venv, 此处默认安装的是系统的python版本, 指定Python版本使用virtualenv venv python=python3.6)
witch python (查看当前python的路径)
. /venv/bin/activate (激活虚拟环境)
deactivate (新版停用命令, 旧版命令:. /venv/bin/deactivate)
pip install flask (在虚拟环境中安装包, 若授权失败, 解决方案: sudo chown -R username(用户名) /home/workspace/flask_demo1/venv/(虚拟环境路径), 注意如果使用sudo安装的话, 安装的将会是全局状态的包)
pip list (查看当前环境安装的所有包)
touch requirements.txt
pip freeze > requirements.txt (记录虚拟环境安装的依赖, 迁移环境重置使用： pip install -r requirements.txt)
