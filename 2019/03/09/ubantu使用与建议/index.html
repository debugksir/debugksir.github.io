ubantu 使用与建议

  • 使用环境： vmware 虚拟机
  • 若硬件条件有限建议使用 <= 16版本的ubantu desktop版本
  • 汉化： 打开设置（按下super键后输入setting）=> 语言支持 => 添加或删除语言 => 下载chinese => 拖动汉语到最上层 => ok
  • 中文输入法： 到搜狗官网下载搜狗输入法linux版，完了双击安装包进行安装，打开设置 => 文本输入 => 配置输入法。 另外语言支持的键盘输入系统选择fictx
  • 个人开发装机(sublime, virtual studio code, chrome, mysql, mysql-workbench, node.js)
